◆ USLocalization()

void USLocalization ( )
inline

This method performs the Ultrasonic localization using a falling edge detection.

It makes the robot turn clockwise until a falling edge is detected, record the odometer's angle, switch direction then detects a second falling edge and record the odometer's angle. Then, it uses those two angles to turn to the 0° direction.
