◆ attempt()

boolean attempt ( )
inline

This method checks for a ring in the side that the robot is facing.

It makes the robot go close to the tree, then starts scanning for a ring while making the sensor move up and down. If a ring is detected, the robots grab it and beeps according to the ring's color and backs of. Otherwise, the robot just backs of.

Returns
boolean: true when a ring is detected and grabbed, false otherwise.
