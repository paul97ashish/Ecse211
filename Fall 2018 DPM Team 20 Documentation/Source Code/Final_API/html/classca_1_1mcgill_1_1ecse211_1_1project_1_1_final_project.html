◆ TraverseTunnel()

static void TraverseTunnel ( )
inlinestatic

This method make the robot go close to the tunnel, traverse it and localize at the other end of the tunnel.

We are using the array "orientation" to get both the orientation of the tunnel and the direction of the traversal.

orientation[0] gives the orientation of the tunnel (true means vertical and false means horizontal).

orientation[1] gives the direction of the tunnel.For the vertical case: true is UP and false is DOWN. For the horizontal case: true is LEFT and false is RIGHT.
